# Project to Rspec practice

## install

```bash
$ bundle install
```

## Running tests

### Simple running

```bash
$ rspec
```

### Run and show suites names with contexts

```bash
$ rspec -fd
```

### Generate HTML reporting

```bash
$ rspec --format html --out report.html
```

### Generate Junit reporting format

```bash
$ rspec --format RspecJunitFormatter --out report.xml
```
